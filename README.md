# Skinbiome - Como abrir o projeto na web

Este projeto roda localmente com Vite.  
Siga os passos abaixo para abrir no navegador.

## Pre-requisitos

- [Node.js](https://nodejs.org/) 18+ (recomendado 20+)
- npm (ja vem com o Node.js)

## 1) Instalar dependencias

No terminal, dentro da pasta do projeto:

```bash
npm install
```

## 2) Iniciar o projeto

```bash
npm run dev
```

## 3) Abrir no navegador

Quando o servidor iniciar, acesse:

- `http://localhost:3000` (porta mais comum neste projeto)

Se a porta estiver ocupada, o terminal mostrara outra URL local.  
Basta abrir a URL exibida.

## Comandos uteis

```bash
npm run build    # gera build de producao
npm run preview  # abre a build localmente
npm run lint     # roda o eslint
npm run format   # roda o prettier
```
